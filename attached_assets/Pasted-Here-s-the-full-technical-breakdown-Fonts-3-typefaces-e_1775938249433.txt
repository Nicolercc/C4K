Here’s the full technical breakdown:

Fonts
3 typefaces, each with a specific job:
1. Poppins — font-family: 'Poppins', sans-serif

Loaded from Google Fonts with weights: 400, 500, 600, 700, 800, 900
Used via Tailwind class font-poppins
Role: All headings, titles, buttons, labels, XP numbers, nav items — anything that needs to feel bold and authoritative
Why: Poppins is a geometric sans-serif with perfectly circular letterforms. At font-900 it hits hard and reads great at large sizes. Very popular in edtech/gaming UI
Example usage:font-poppins font-800 text-4xl on the hero H1, font-poppins font-700 on all buttons
2. Quicksand — font-family: 'Quicksand', sans-serif

Loaded with weights: 400, 500, 600, 700
Used via font-quicksand
Role: Body copy, descriptions, subtitles, footer text, card paragraphs — anything conversational
Why: Quicksand is a rounded sans-serif with soft terminals (the ends of strokes are rounded, not sharp). This makes it feel friendly and approachable — perfect for kids reading instructions
Example usage:font-quicksand text-lg text-gray-500 on section subtitles
3. JetBrains Mono — font-family: 'JetBrains Mono', monospace

Loaded with weights: 400, 500, 600
Used via font-mono
Role: The code editor panel, syntax-highlighted code, the floating particle symbols in the hero
Why: JetBrains Mono is a purpose-built coding font with wide letter spacing and clear distinction between similar characters (0 vs O, l vs 1). It also has ligatures. Kids reading code need maximum clarity
Example usage: The lesson editor’s center panel, <h1> tags in the floating particles

Color application strategy:

Hero overlay:from-slate-900/75 via-indigo-950/60 to-teal-950/70 — deep navy with 60–75% opacity sitting over the neon city image
Particle colors cycle through: #2DD4BF (teal), #a78bfa (soft purple), #FFD93D (yellow), #FF6B9D (pink), #ffffff (white)
Semantic color logic: teal = success/progress, yellow = rewards/XP, pink = lives/hearts, purple = evolution/power, coral = errors/mistakes
TL;DR: Poppins punches, Quicksand chats, JetBrains codes. The palette is a neon-candy system — high saturation primaries on dark overlays, pastel tints on light backgrounds. Very deliberate split between “game UI” colors (yellow, teal, pink) and “reading comfort” neutrals (gray-500, white, violet-50).
